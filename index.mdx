    </Step>
    
    <Step title="Configure your tests">
      Update the base URL in `cypress.config.js` to point to your application.
      
      ```javascript
      module.exports = defineConfig({
        e2e: {
          baseUrl: 'https://your-app.com/',
          viewportWidth: 1920,
          viewportHeight: 1080
        }
      });
      ```
    </Step>
    
    <Step title="Run your first test">
      Open Cypress Test Runner or execute tests in headless mode.
      
      ```bash
      npx cypress open
      ```
      
      For headless execution:
      
      ```bash
      npm test
      ```
    </Step>
